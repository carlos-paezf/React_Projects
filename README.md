## Initial Setup

- Create project:

    ```txt
    $: pnpm create vite {00-Project_Name} --template react-ts
    ```

- Install project libraries:

    ```txt
    $: pnpm install
    ```

- Execute project in dev mode:

    ```txt
    $: pnpm dev
    ```

- Build deploy project:

    ```txt
    $: pnpm build
    ```
